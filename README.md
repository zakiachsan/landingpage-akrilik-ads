# Landing Page Akrilik Ads

Landing page high-conversion untuk iklan frame akrilik Pak Juli.

## Fitur Landing Page

- Hero section dengan CTA kuat
- Problem agitation (frame telat, salah produk, biaya siluman)
- 3 varian produk dengan detail bahan dan kelengkapan
- Use cases (Rumah sakit, hotel, rumah makan, kantor)
- Social proof & testimoni
- Added value (free sample, garansi, dll)
- FAQ accordion
- Strong final CTA

## Struktur Produk

### 1. Full Akrilik Premium - Rp 450.000
- Bahan: Akrilik bening premium grade A
- Termasuk: 4 baut stainless, 4 skrup, kotak packaging

### 2. Depan Akrilik + MDF - Rp 350.000
- Bahan: Depan akrilik, belakang MDF solid
- Termasuk: 4 baut stainless, 4 skrup, kotak packaging

### 3. Full PVC Economy - Rp 250.000
- Bahan: PVC premium dengan double tape
- Termasuk: 4 pcs double tape 3M, kotak packaging

## Cara Menjalankan

```bash
# Install dependencies
npm install

# Jalankan development server
npm run dev

# Build untuk production
npm run build
```

## Output

Static export akan tersimpan di folder `dist/` siap untuk di-deploy.
